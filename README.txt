TM2-xaseco2-plugin for showing a custom laps-counter

INSTALLATION:
1) put the c_lapscounter.php-file in the xaseco2/plugins-folder
2) in the xaseco2/plugins.xml add the line:
   <plugin>plugin.c_lapscounter.php</plugin>
3) put the c_lapscounter.xml-file in the xaseco2-folder
   in this file you can
   - adapt the position and size of the counter
   - indicate for which gamemode it should work
   - indicate if it should take the numberOfLaps of the matchsettings or of the map itself
4) restart xaseco2
